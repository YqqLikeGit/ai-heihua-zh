# Tab 补全

> **Tab Completion** · #cursor

## 一句话

按 Tab 接受 AI 生成的代码建议

## 打个比方

输入法联想但整段代码

## 为什么重要

- 面试、读文档、看 GitHub 项目时会反复遇到
- 搞懂黑话 = 少踩坑、少被 AI 忽悠
