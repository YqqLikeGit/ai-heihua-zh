# Vibe Coding 会取代程序员吗？

> 5 分钟读完

## 什么是 Vibe Coding？

**用自然语言描述「感觉 / 意图」，让 AI 写大部分代码。**

你：「做一个能查 GitHub Star 的 CLI，要酷一点」  
AI：生成项目结构、脚本、README。

## 会取代程序员吗？

**短期：不会。长期：会取代「只会抄模板」的 coder。**

| 仍需要人 | AI 更擅长 |
|----------|-----------|
| 定需求、验收 |  boilerplate |
| 架构与安全 | 重复 CRUD |
| 背锅与沟通 | 搜 API 用法 |

## 怎么用好 Vibe Coding？

1. **说清楚约束** — 语言、框架、不能做什么
2. **小步迭代** — 不要一次「做个淘宝」
3. **会读 diff** — 不懂 diff 就不要点 Accept All
4. **Rules / Skills** — 让 AI 遵守项目规范
