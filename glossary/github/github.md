# GitHub

> **GitHub** · #github

## 一句话

全球最大的代码托管与开源协作平台，程序员的「代码朋友圈」

## 打个比方

像微博+网盘，但存的是代码和版本历史

## 为什么重要

- 面试、读文档、看 GitHub 项目时会反复遇到
- 搞懂黑话 = 少踩坑、少被 AI 忽悠
