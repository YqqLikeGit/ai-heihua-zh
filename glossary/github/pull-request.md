# Pull Request / PR

> **Pull Request** · #github

## 一句话

请求把你的改动合并进原项目

## 打个比方

给原作者发「我改好了，请收」的合并申请

## 为什么重要

- 面试、读文档、看 GitHub 项目时会反复遇到
- 搞懂黑话 = 少踩坑、少被 AI 忽悠
