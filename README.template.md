## 终端查词

```bash
git clone https://github.com/YqqLikeGit/ai-heihua-zh.git
cd ai-heihua-zh
./bin/heihua star          # 查「Star 是什么」
./bin/heihua mcp           # 查 MCP
./bin/heihua github        # 查 GitHub
./bin/heihua --list ai     # 列出 AI 分类全部词条
```
