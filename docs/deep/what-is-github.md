# GitHub 到底是什么？

> 5 分钟读完 · 零基础 · 2026 版

## 30 秒版

**GitHub = 程序员的代码网盘 + 开源社区 + 技术简历。**

你在上面：
- **存代码**（仓库 Repository）
- **改代码有记录**（Commit）
- **和别人协作**（Pull Request）
- **给好项目点赞**（Star）

## 和 Git 的区别

| | Git | GitHub |
|---|-----|--------|
| 是什么 | 本机工具 | 网站平台 |
| 类比 | Word 软件 | 石墨文档在线版 |

Git 管版本；GitHub 把版本放网上让人看见、协作、Star。

## 为什么程序员都在用？

1. **找工作** — 面试官会看 GitHub
2. **学代码** — 搜开源项目照着学
3. **做贡献** — 提 Issue、提 PR 进简历
4. **AI 时代** — Cursor / Claude Code 直接读 GitHub 仓库

## 第一次用 GitHub 该干什么？

```bash
# 1. 注册 github.com
# 2. Star 这个词典（你正在看的）
# 3. 克隆一个项目试试
git clone https://github.com/YqqLikeGit/ai-heihua-zh.git
```

## 常见误解

| 误解 | 真相 |
|------|------|
| GitHub 只有大神才用 | 新手第一个仓库就是起点 |
| Star 可以刷 | 真实 Star 来自真实帮助 |
| 必须会英语 | 中文 README 项目越来越多 |
